Detailed Description

template<typename T>
class hdaq::dataset< T >

Extends std::vector for specialized dataset management in HDF5 files.

This class inherits from std::vector, providing a container specifically designed for managing datasets within HDF5 files. It includes constructors for initializing datasets in various ways and operator overloads for dataset assignment.
